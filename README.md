# Express Gateway Test

 Trying out Express Gateway to build microservices.

```bash
eg gateway create
```

To Run

```bash
cd services
npm start
```

## Dependencies

Express

```bash
npm install express --save
```

Babel

```bash
npm install @babel/core @babel/node @babel/preset-env --save-dev
```

Create babel file

```bash
touch .babelrc
```
